How to Apply CSS to HTML

Use of an external CSS stylesheet is permitted using a <link> element with an href attribute; however, the file must be within the directory. An example is: 

	<link rel="stylesheet" href="styles.css"/>
			
	The contents of styles.css could include: 
			
	p {
		color: white;
		background-color: gray;
		border: 1px solid black;
		padding: 5px
	}

How to Use JavaScript in a Webpage

  • Linking an external JavaScript file:

    • Similar to CSS, a JavaScript file can be written in its own .js file and can be executed using <script> with a src attribute pointing to the script file. 

    	<script src="path/to/my/script.js"> </script/>

  • Writing JavaScript within an html file:

    • JavaScript code can be added between the <script> tags instead of using a src attribute. 

    	<script> 
		window.addEventListener("load", () => {
		  console.log("This function is executed once the page is fully loaded");
		});
	</script/>
